Хатан
Мэргэд аймгаас хатнаа буцаан авахын тулд Тэмүжиний толгойлсон боржигин овгийн x тооны цэргүүд нэгэн ууланд иржээ. Мэргэдүүд ууланд нүх ухаж n өрөөг үүсгэсэн ба тэдгээр нь хоорондоо аль ч чиглэлд явж болох m тооны хонгилоор холбогдсон. Боржигинчууд эхлээд 1-р өрөөнд орж ирэх ба хатан n-р өрөөнд байгаа.
i-р өрөөнд ai тооны мэргэдүүд байна. Боржигинчууд орж ирэхэд тулаан эхэлдэг. Хэрэв боржигинчуудын тоо нь мэргэдүүдийн тооноос их бол бүх мэргэдүүд алагдаж, боржигинчууд бүгд амьд үлддэг. Эсрэг тохиолдолд бүх мэргэдүүд алагдах боловч боржигинчуудын тал нь амь үрэгддэг: хэрэв тулааны эхэнд x тооны боржигинчууд байсан бол ⌊ x/2⌋ (доод бүхэл тоо) боржигинчууд амьд үлдэнэ. Боржигинчууд очсон өрөө бүртээ мэргэдүүдтэй тулалдах ба 1-р болон n-р өрөөнд мөн тулалдана.
1-р өрөөнөөс n-р өрөө хүртэл хамгийн ихдээ хэдэн боржигинчууд очиж чадахыг ол.
Оролт
Эхний мөрөнд мэргэдүүдийн өрөөний тоо болон хонгилуудын тоо болох n,m бүхэл тоонууд өгөгдөнө (1≤n,m≤2⋅10^5)
Дараагийн m тооны мөрөнд хонгилуудыг тодорхойлно: i-р хонгил нь ui ба vi тоонуудаар тодорхойлогдох ба тэдгээр нь уг хонгилоор хоорондоо холбогдож буй өрөөнүүдийн дугаар юм (1≤ui, vi<=n, ui!=vi). Хос өрөө бүр хоорондоо нэгээс олонгуй хонгилоор холбогдох нь баталгаатай байна.
Дараагийн мөрөнд x тоо байх ба энэ нь 1-р өрөө рүү орж байгаа боржигинчуудын тоо юм (1≤x≤10^9).
 Сүүлийн мөрөнд a1,a2,...,an тоонууд байрлах ба эдгээр нь өрөөнүүд дэх мэргэдүүдийн тоо юм (1≤ai ≤10^9 ).
 
Гаралт
1-р өрөөнөөс эхлээд n-р өрөөнд очиж чадах боржигинчуудын хамгийн их тоог гаргана. Ядаж нэг боржигин хүн амьд үлдэх нэг ч аялал олдохгүй бол 0-ийг гаргана.
