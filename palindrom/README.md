## Давхар палиндром
Палиндром гэдэг нь урдаасаа болон ардаасаа ижилхэн уншигддаг тэмдэгт мөр юм.
Давхар палиндром гэж өөрөө палиндром байх эсвэл хоёр палиндромыг (хоорондоо ялгаатай байх албагүй) залгахад үүссэн тэмдэгт мөрийг нэрлэнэ.
Жишээ нь roor, pabba болон otto нь давхар палиндром мөн ба rpurh болон huhhuu нь биш юм.
---
Англи цагаан толгойн эхний k үсгээс бүтсэн, дээд тал нь n урттай, хоосон биш тэмдэгт мөрүүдээс хэд нь давхар палиндром байхыг ол.
Энэ тоог 998244353 тоонд хуваасны үлдэгдлийг хэвлэж гаргана.
Оролт
Тэмдэгт мөрийн хамгийн их урт ба цагаан толгойн үсгийн тоог илэрхийлэх n, k бүхэл тоонууд байрлана.
(1 ≤ n ≤ 10^5, 1 ≤ k ≤ 26)
Гаралт
