# Бодлого: Палиндром

Палиндром гэдэг нь урдаасаа болон ардаасаа ижилхэн уншигддаг тэмдэгт мөр юм.

### Давхар палиндром
Давхар палиндром гэж өөрөө палиндром байх, эсвэл хоёр палиндромыг (хоорондоо ялгаатай байх албагүй) залгахад үүссэн тэмдэгт мөрийг нэрлэнэ. 

* **Жишээ нь:** `popanna`, `pabba` болон `otto` нь давхар палиндром мөн.
 `ppypx` болон `xyxxyy` нь давхар палиндром биш юм.

### Даалгавар
Англи цагаан толгойн эхний k үсгээс бүтсэн, дээд тал нь n урттай, хоосон биш тэмдэгт мөрүүдээс хэд нь давхар палиндром байхыг ол. Энэ тоог **998 244 353** тоонд хуваасны үлдэгдлийг хэвлэж гаргана.

### Оролт
Тэмдэгт мөрийн хамгийн их урт ба цагаан толгойн үсгийн тоог илэрхийлэх n, k бүхэл тоонууд байрлана. $$(1 \le n \le 10^5, 1 \le k \le 26)$$

### Гаралт
Англи цагаан толгойн эхний k үсгээс бүтсэн, дээд тал нь n урттай, хоосон биш, давхар палиндром тоонуудыг (мөрүүдийг) тоолж, **998 244 353** модулиар хэвлэнэ.

### Жишээ тест
| Оролт (n, k) | Гаралт (Үлдэгдэл) |
| :--- | :--- |
| `2 2` | `6` |
| `5 2` | `62` |
| `32 8` | `527120306` |

**Тайлбар:** Эхний жишээ ($n=2, k=2$) дээр `a, b, ab, ba, aa, bb` гэсэн зургаан давхар палиндром олдоно.
